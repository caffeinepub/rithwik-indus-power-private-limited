# Rithwik Indus Power Private Limited - Frontend

This is the frontend application for Rithwik Indus Power Private Limited, a solar energy company based in Tamil Nadu, India.

## Overview

A modern, responsive single-page application showcasing the company's 5MW solar power project in Madurai, Tamil Nadu. Built with React, TypeScript, and Tailwind CSS, deployed on the Internet Computer.

## Features

- **Responsive Design**: Optimized for all devices from mobile to desktop
- **Solar Energy Theme**: Custom OKLCH color system with deep forest green and solar yellow accents
- **Smooth Navigation**: Single-page application with smooth scroll navigation
- **Contact Information**: Easy access to company details and phone contact
- **Sustainability Focus**: Highlighting environmental benefits and green initiatives

## Tech Stack

- **Framework**: React 19 with TypeScript
- **Styling**: Tailwind CSS with custom solar energy theme
- **UI Components**: Shadcn/ui (Radix UI primitives)
- **Icons**: Lucide React
- **Backend**: Internet Computer (Motoko)
- **State Management**: React Query (TanStack Query)
